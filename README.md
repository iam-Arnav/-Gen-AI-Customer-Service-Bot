# Projects

## Task 1 – Sentiment Analysis Chatbot

Integrated sentiment analysis into a customer support chatbot to recognize user emotions and generate responses that are better aligned with the customer's emotional state.

**Highlights**
- Positive, Negative & Neutral sentiment detection
- Emotion-aware responses
- Transformer-based sentiment analysis
- Interactive Streamlit application

---

## Task 2 – Medical Q&A Chatbot

Developed a medical question-answering chatbot using the MedQuAD dataset with retrieval-based response generation and basic medical entity recognition.

**Highlights**
- Medical Question Answering
- MedQuAD dataset integration
- Retrieval-Augmented Generation (RAG)
- Medical entity recognition
- Streamlit interface

---

## Task 3 – Dynamic Knowledge Base Expansion

Built a chatbot capable of automatically expanding its knowledge base by updating the vector database with newly added documents, allowing it to answer questions using the latest available information.

**Highlights**
- Dynamic document ingestion
- Automatic FAISS index updates
- Continuous knowledge expansion
- Semantic search
- Streamlit dashboard

---

## Task 4 – Domain Expert Chatbot

Created a domain-specific chatbot trained on research papers from the arXiv dataset. The chatbot can answer technical questions, summarize research papers, explain complex concepts, and handle follow-up discussions.

**Highlights**
- Scientific paper search
- Research paper summarization
- Technical concept explanation
- Follow-up conversations
- Domain-specific AI assistant

---

## Task 5 – Multi-Modal AI Assistant

Designed a multimodal AI assistant capable of understanding both text and image inputs. The assistant performs contextual reasoning, maintains conversation history, and generates evidence-based responses.

**Highlights**
- Image understanding
- Visual Question Answering
- Context-aware reasoning
- Response validation
- Multi-turn conversational memory

---

## Task 6 – Multilingual AI Chatbot

Extended the chatbot to support multilingual conversations while preserving context and intent across language switches. The assistant automatically detects languages and handles mixed-language interactions seamlessly.

**Highlights**
- Automatic language detection
- Multilingual conversations
- Mixed-language support
- Cross-lingual reasoning
- Context preservation

# Getting Started

### Clone the repository

```bash
git clone https://github.com/iam-Arnav/-Gen-AI-Customer-Service-Bot.git
```

### Navigate to any project

```bash
cd "Task 1 - Sentiment Analysis Chatbot"
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install the required dependencies

```bash
pip install -r requirements.txt
```

### Configure the API Key

Create a `.env` file inside the project folder:

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

### Run the application

```bash
streamlit run main.py
```
